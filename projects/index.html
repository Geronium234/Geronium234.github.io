Detective in Distress

Unity 2D Visual Novel C#

Overview

Detective in Distress is a 2D visual novel where players take on the role of a detective trying to solve a missing person case by travelling to different times.

My Role

I was responsible for three main areas of development:

  • Story slider system — programmed the dialogue and scene-progression system that drives the visual novel flow, handling branching, character expressions, and timed text reveals.
  • 2D platforming — built the movement and collision systems used in the platforming segments that act as transitions between story beats.
  • Art integration — imported, sliced, and animated all 2D art assets in Unity, including character sprites, backgrounds, and UI elements.

Kaboom

Godot 3D FPS GDScript

Overview

A personal project my friend and I built in Godot. Kaboom is a 3D FPS built around the idea of combining different bullets from other weapons. We made an intro level to showcase the mechanic.

My Role

I was responsible for programming the core gameplay mechanics:

  • Animation system — rigged and implemented first-person weapon animations including idle, fire, and reload states using Godot's AnimationPlayer and AnimationTree.
  • Shooting system — designed and programmed the bullet-combination mechanic, including projectile types, collision detection, and effect application on enemies and the environment.
  • Level design — blocked out and dressed the intro level, pacing the player's discovery of each bullet type so the combination system is learned naturally through play.

Skyrise Frictionless Ascent

Unity 2D Vertical Platformer

Overview

Skyrise Frictionless Ascent is a vertical platformer where players control a character with no friction, making movement and momentum the key to navigating the level. I was responsible for creating everything in this project.

My Role

Solo project — I handled every aspect of development:

  • Physics-based controls — programmed the frictionless movement system, tuning momentum, gravity, and wall interactions to make the lack of friction feel intentional and fun rather than frustrating.
  • Level design — designed and built the vertical level layout to teach the player the movement system progressively, introducing new obstacles as momentum mastery grows.
  • Audio — sourced and implemented all sound effects and music, handling audio triggers and looping in Unity.
  • Animations — created and integrated all character and environment animations.
